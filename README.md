# CoreIR - Self-Learning Intermediate Representation

A universal runtime layer for software that learns and improves itself.

## Primitives
- **diff** — automatic differentiation for gradient-based learning
- **search** — probabilistic exploration over program space
- **rewrite** — safe runtime self-modification with proof verification

Requires LLVM/MLIR. See CMakeLists.txt.
